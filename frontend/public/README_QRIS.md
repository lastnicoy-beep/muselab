# QRIS Image Setup

Untuk menampilkan QRIS code di halaman payment:

1. Simpan gambar QRIS Anda dengan nama `qris.png` atau `qris.jpg`
2. Letakkan file tersebut di folder `frontend/public/`
3. File akan otomatis tersedia di `/qris.png` atau `/qris.jpg`

**Catatan:**
- Format yang didukung: PNG, JPG, JPEG
- Ukuran disarankan: minimal 512x512px untuk kualitas scan yang baik
- Pastikan QR code jelas dan tidak blur

Setelah file diletakkan, QRIS akan otomatis muncul di halaman payment ketika user memilih metode pembayaran QRIS.
